01｜项目总览：打造“扫地机器人购买决策型内容资产”

扫地机器人已经从“会不会扫”进入“是否真正省心”的竞争阶段。2026年下半年内容重点应从单一清洁力，升级为避障、毛发防缠绕、自清洁、上下水、拖地效果、边角覆盖、宠物家庭和长期维护成本的综合决策。

Project Overview
项目目标 建立覆盖新品趋势、真实测试、大促导购、家庭场景和长期复盘的扫地机内容体系。
内容定位 从“参数搬运”升级为“家庭清洁自动化购买顾问”。
核心阵地 B站长视频为主，搭配短切片、动态答疑、合集和双11转化。
商业目标 形成大促榜单、品牌合作、带货转化、测试协议和长期清洁电器栏目。

2026下半年品类判断

  • 全能基站普及：自动集尘、洗拖布、热水洗、热风烘、自动上下水将进一步下探。
  • AI避障进入体验竞争：从“能识别障碍物”变成“能否不误判、不漏扫、不撞宠物用品”。
  • 宠物家庭成为核心人群：毛发防缠绕、猫砂清理、宠物粪便避障、异味控制会成为爆点。
  • 拖地能力继续升级：滚筒拖、机械臂外扩拖、加压拖、热水复洗等概念需要实测验证。
  • 长期维护成本重要性上升：耗材、清洗、管路、基站异味、售后维修将影响用户口碑。

半年内容目标

  • 搜索资产：沉淀“扫地机器人怎么选、扫拖一体、自动上下水、宠物家庭扫地机、双11扫地机推荐”等关键词。
  • 测试资产：建立统一测试协议，覆盖灰尘、颗粒、毛发、湿污、边角、越障、避障、噪音、基站和维护成本。
  • 商业资产：形成品牌合作案例、大促榜单、带货转化数据和栏目报价能力。
  • 信任资产：把“真实家庭地面测试”和“长期使用复盘”变成账号差异化。

07｜扫地机器人科学测试体系

扫地机测评最容易失真的是“测试太干净、场景太理想、只测一次”。建议采用统一标准，覆盖清扫、拖地、避障、基站和长期维护。

Testing Protocol
测试维度 关键指标 测试方法 适用选题 结论表达
干垃圾清扫 灰尘、米粒、猫砂、咖啡渣、边角残留 固定克重、固定地面、固定路线，清扫前后称重和拍照。 入门横评、双11推荐 说明硬地板、地毯、边角表现差异。
毛发防缠绕 长发、宠物毛、滚刷缠绕、边刷缠绕 用定量长发和宠物毛反复测试，记录清理难度。 养宠家庭、长期体验 不要只看“防缠绕宣传”，要看实际拆洗频率。
湿污拖地 酱油、咖啡、脚印、厨房油污、水渍 设置新鲜污渍和干涸污渍，比较一次拖净率和复拖效果。 拖地专项、旗舰横评 区分“日常维持清洁”和“深度清洁”。
边角覆盖 墙边、桌腿、柜边、门后、基站附近 撒粉或颗粒观察贴边轨迹和残留区域。 全机型横评 机械臂、外扩拖布、算法贴边要分别验证。
避障能力 线缆、拖鞋、袜子、宠物碗、玩具、粪便模型 布置典型家庭障碍物，记录误撞、漏扫、缠绕和绕行。 AI避障、新品趋势 识别成功不等于清洁体验好，需看绕行后是否漏扫。
导航建图 建图速度、分区准确、地图稳定、禁区设置 多房间、多门槛、多家具环境反复运行。 长期体验、装修入住 重点说明复杂户型和家具变动后的表现。
越障与地毯 门槛、地垫、地毯识别、拖布抬升 设置不同高度门槛和不同材质地毯测试。 装修预留、家庭场景 明确适合平层、复杂户型还是地毯家庭。
基站能力 集尘、洗拖布、烘干、自清洁、上下水 连续运行7-30天，记录异味、残水、污水箱和维护频率。 全能基站、长期复盘 核心是“省心程度”，不是功能数量。
噪音体验 清扫噪音、集尘噪音、烘干噪音 用噪音仪在固定距离测试不同模式。 母婴、宠物、居家办公 说明白天、夜间、家有宠物/婴儿的使用边界。
长期成本 耗材、清洁液、尘袋、滤芯、滚刷、售后 统计半年/一年耗材价格和更换频率。 双11推荐、年度红黑榜 给出“买得起”和“养得起”的完整结论。
测试边界声明： 每期视频建议说明测试地面、污渍类型、样本克重、运行模式、固件版本、样机来源和结论适用范围。

08｜长视频制作流程与项目门禁

扫地机器人长视频样机多、测试脏、周期长，建议每期提前10-14天立项，横评类提前20天准备。

Production SOP
1立项明确用户、节点、预算档和核心问题。
2调研收集搜索词、电商差评、竞品视频和品牌资料。
3样机确认自购、借测、送测或商业合作。
4测试按统一协议记录清洁、避障、基站和长期表现。
5脚本前置结论，按用户画像解释测试结果。
6拍摄拍污渍布置、运行过程、残留、基站和维护。
7剪辑章节、数据卡、对比表、封面和标题。
8发布合集、时间轴、置顶评论、价格更新和答疑。

Gate 1：选题门禁

没有明确用户场景和可测试变量，不进入制作。

Gate 2：测试门禁

污渍克重、地面、路线、模式不统一，不输出强对比结论。

Gate 3：商业门禁

品牌合作不得干预测试结果，必须保留缺点和不适合人群。

Gate 4：发布门禁

标题封面、样机声明、价格说明、置顶评论、合集未完成不发布。

09｜发布运营与数据增长打法

扫地机器人内容兼具传播和转化属性。宠物、脏污测试适合传播；双11榜单、价格监测适合转化；长期复盘适合建立信任。

Operation
运营阶段 关键动作 重点指标 优化方法
发布前 动态征集宠物、户型、预算、地面材质和清洁痛点。 评论问题质量、预约互动。 用“面积+宠物+预算+是否上下水”收集需求。
发布0-6小时 置顶核心结论，快速回复预算和型号问题。 点击率、前30秒留存、评论率。 点击低改封面标题；留存低优化开头切片。
发布1-7天 剪短切片：毛发测试、避障翻车、拖地对比、基站维护。 二跳播放、收藏率、合集访问。 高传播片段回流长视频。
大促期间 更新置顶价格，补充型号变动，配合橱窗/直播答疑。 带货点击、评论咨询、转化率。 价格必须标注更新时间，避免过期误导。
